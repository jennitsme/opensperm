# Engineering Principles
- First-principles design; avoid cargo cult.
- Typed everywhere; strict schemas for IO.
- Secure-by-default: isolation, least privilege, approvals for sensitive scopes.
- Deterministic dev loop: fixtures, golden transcripts, reproducible runs.
- Observability first: traces, logs, metrics per tool call.
- Modular + extensible: SDKs with middleware; capability graph for composition.
